# 퇴근 후 콘텐츠 추천 시스템 (RAG 기반)

사용자의 기분, 가용 시간, 선호 플랫폼에 맞추어 퇴근 후 볼만한 콘텐츠를 RAG(검색 증강 생성) 기반으로 추천해 주는 시스템입니다. 

## 프로젝트 문서 안내

이 프로젝트는 원활한 개발 및 AI 어시스턴트와의 협업을 위해 체계적인 문서 구조를 가지고 있습니다. 프로젝트의 상세한 내용은 아래 문서들을 참고해 주세요.

- [시스템 아키텍처 및 설계](./docs/architecture/system-design.md) : 기술 스택, 인프라 구성, 데이터 파이프라인 흐름
- [개발 로드맵 (Roadmap)](./docs/roadmap.md) : 프로젝트 개발 순서 및 마일스톤
- [RAG 기반 추천 기능 상세](./docs/features/recommendation.md) : 기능 시나리오 및 예외 처리 명세
